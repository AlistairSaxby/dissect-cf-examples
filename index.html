Welcome to DISSECT-CF examples.

These pages are intended to help you get into the depths of the simulations possible with DISSECT-CF. The sources provided here are aimed at showing more complete and complex use cases of the simulator. Despite the complex use cases, the examples project was designed with simplicity in mind to allow a gentle but detailed introduction to the concepts of the simulator.

When using the simulator for scientific research please cite the following paper:

  • Gabor Kecskemeti: DISSECT-CF: A simulator to foster energy-aware scheduling in infrastructure clouds. In Simulation Modelling Practice and Theory. 2015. DOI: 10.1016/j.simpat.2015.05.009

The API documentation

The javadoc API of this project shows the intentions of each command line utility, as well as the intended behavior of the complex use cases for the simulator. Although the javadocs are already useful for introductory material, it is recommended to read the in line comments in the sources of this project as well to get a better grasp of the simulator's usage.

For past project releases an archive for the API can be found here.

Support or Contact

Apart from the documentation listed here, one can find plenty of useful information about the way the simulator is intended to work at the simulator's homepage. In case you still have some questions, please share them with the simulator's user and developer community at our Q&A forum - a registration is needed to send your questions in.
